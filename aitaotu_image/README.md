### 描述
使用scrapy爬取([https://www.aitaotu.com/](https://www.aitaotu.com/))图片  
信息保存至MongoDB、MySQL，图片保存至根目录
### 关键字
python、scrapy、MongoDB、MySQL、Images Pipeline
### 环境/依赖
win7/python3
### 运行
scrapy crawl aitaotu
